mecanicas do game

Mecanismos são sistemas de blocos usados para realizar certas tarefas, como abrir uma porta de longe ou revelar uma escada escondida. Esses sistemas são construídos a partir de componentes simples e normalmente envolvem algum tipo de entrada do usuário, como quebrar uma tocha, o que gera algum tipo de resultado, como uma porta sendo revelada. Os mecanismos podem variar de simples interruptores que abrem e fecham portas a uma distância segura, a dispositivos complexos, como fechaduras de combinação que evitam que intrusos entrem em seu forte.
